# Bank Management System

## Project Description

Bank Management System is a console-based application developed in C++ using Object-Oriented Programming (OOP) concepts and File Handling. The system allows users to manage bank accounts and perform basic banking operations efficiently.

## Features

* Create New Account
* Deposit Money
* Withdraw Money
* Check Account Balance
* Display All Accounts
* Store Customer Records Using File Handling

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling
* VS Code

## How to Run

1. Compile the program:

g++ main.cpp -o bank

2. Run the program:

./bank

## Expected Outcome

The application provides a secure and efficient way to manage customer bank accounts. It supports deposits, withdrawals, balance inquiries, and persistent storage of account information.
